Computes Cp by 3rd-order polynomial data fit: cp(T) = c1*T^3 + c2*T^2 + c3*T + c4

Coefficients (with 95% confidence bounds): c1 = -7.357e-007 (-9.947e-007, -4.766e-007) c2 = 0.001307 (0.0009967, 0.001617) c3 = -0.5558 (-0.6688, -0.4429) c4 = 1074 (1061, 1086)

Assumptions:

123 K < T < 673 K
